四、开局随机背景
系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：被控制】

背景故事：过去半年你频繁在{TA}提出“去 XX 吧”的时候问“安全吗”“有没有规划”“你确认过吗”。{TA}回答过几次，后来就不再问了。{TA}发现自己“被规划”了——{TA的}冲动需要先通过你的“审批”。{TA}开始把那些冲动留给自己。

S-01特殊旁白：“{TA的}手机里有一个备忘录，叫‘想去的地方’。最近一条是上个月加的。没有划掉——说明还没有去。{TA}以前会直接去，不写备忘录。”

【裂痕类型二：被评价】

背景故事：你曾经在{TA}兴奋地分享“我们可以去做 XX”的时候说“你怎么总想些不靠谱的事”。{TA}没有反驳，但之后{TA}不再跟你分享那些“可能不靠谱”的计划了。{TA}只跟你分享那些“确定了、安全的、已经验证过”的事。

S-01特殊旁白：“{TA}今天说了一句‘我最近在想……’然后看了你一眼——然后说‘算了，没事’。{TA}把‘在想’的那个版本收回去了。”

【裂痕类型三：被忽视】

背景故事：{TA}曾经带你去过很多“临时的、没有计划的”地方。你去了，但你大部分时间都在看手机。{TA}后来不再带你去了。不是生气了——是觉得你的注意力不在“那个地方”，也不在“和{TA}一起”。

S-01特殊旁白：“{TA}最近开始一个人出门了。出门前{TA}会说‘我出去一下’，然后不说是去哪。不是隐瞒，是觉得‘说了你也不会注意’。”

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"] && flags["Anchor-06"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
