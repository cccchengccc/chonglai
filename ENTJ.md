系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：被抵抗】

背景故事：过去半年你频繁抗拒{TA的}“安排”——{TA}觉得对你好的事、帮你做的计划、推着你往前走的方向。你没说“不好”，但你说“你别管我了”。{TA}后来真的“不管”了——不是不爱了，是学会了“不主动介入你的决定”。

【裂痕类型二：被看弱】

背景故事：你曾经在关键决策上对{TA}说“你决定吧，我都行”。{TA}当时没说什么，但后来{TA}开始不再征求你的意见了。因为“都行”在{TA的}系统里被记录为“{TA}没有判断力”或“{TA}懒得参与”。

S-01特殊旁白：“{TA}今天做了一个决定——关于你们下个月的一个安排。{TA}做了。然后告诉了你。没有问你的意见。不是故意忽略，是{TA}已经不再预设你会有意见。”

【裂痕类型三：被辜负】

背景故事：你曾经在{TA}帮你规划了一件重要事情之后，临时改变主意、没有按计划执行，也没有提前通知{TA}。{TA}花了很长时间消化这件事——不是生气，是重新校准了对你的“可预测性”评估。

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"] && flags["Anchor-06"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
