四、开局随机背景
系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：失约】

背景故事：你连续三次在重要的事情上迟到——{TA}生日你迟到40分钟、{TA}约好一起见朋友你临时放鸽子、{TA}说“今晚我需要你在”你加班到凌晨。{TA}每次都说“没关系，下次注意”。但{TA}开始不再提前和你约任何时间。

S-01特殊旁白：“{TA的}手机日历还是共享状态——但上周的日程里，所有和你有关的条目，都被改成了‘待定’。”

【裂痕类型二：无序】

背景故事：你总是随手乱放东西、忘记关灯、把{TA的}物品挪动后不放回原位。{TA}收拾过很多次，没有说任何重话——但有一天你发现，{TA}把你的东西单独放在了一个抽屉里，和{TA的}物品分开了。

S-01特殊旁白：“你打开衣柜——{TA的}衣服整齐地挂在一侧，你的衣服卷成一团塞在另一侧。中间隔了三个衣架。”

【裂痕类型三：轻诺】

背景故事：你经常随口说“改天我们去看那个”“下次我带你去那家店”——但“改天”和“下次”从来没有实现过。{TA}不再回应你的“改天”，只会在你说完后安静地点一下头。

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"] && flags["Anchor-06"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
