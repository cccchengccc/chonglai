四、开局随机背景
系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：被消耗】

背景故事：过去半年你一直在接受{TA的}照顾，却从没问过“你累不累”。

S-01特殊旁白：“{TA}今天状态不太好，眼睛下有明显的黑眼圈。但你进门时{TA}站了起来：‘你来啦！我今天做了你上次说想吃的那个菜！’”

【裂痕类型二：被神化】

背景故事：你常说“你像天使一样”“你怎么什么都能处理好”。{TA}更努力地“处理好一切”，因为如果不处理好一切，那个“天使”的称号就会掉下来。

S-01特殊旁白：“{TA}今天有一件没有处理好的事——工作上的一个失误。{TA}说这件事的时候语速很快：‘没事，我会解决。你不用担心。’”

【裂痕类型三：被单向】

背景故事：你总是接受{TA的}照顾、{TA的}建议、{TA的}陪伴。但你很少问{TA}“你累不累”。{TA}发现自己在你面前说“累”的时候，你会说“那你休息一下”——但你的语气像在说“那你先去旁边待着，等好了再来”。

S-01特殊旁白：“{TA}今天在你面前打了一个哈欠——然后迅速用手遮住了。{TA}说‘昨晚没睡好而已’，但{TA}遮住哈欠的动作很快，像是怕被看到那个‘也需要睡觉’的版本。”

      return "……我刚才想说‘我今天电量用完了’。但我不确定你会不会觉得我总是在说这个。";
    } else {
      return "……我刚才想跟你说我今天很累。但我怕说了你会觉得‘{TA}又在说累’。";
    }
  }
  // 默认返回
}

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
