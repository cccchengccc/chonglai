四、开局随机背景
系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：被轻看】

背景故事：三个月前你无意中说过“你想得太复杂了”“你总是过度解读”。{TA}没有反驳，只是笑了笑。但从那以后，{TA}不再跟你分享{TA}对“复杂事物”的解读——{TA}学会了用简化的方式和你说话。

S-01特殊旁白：“{TA}今天讲了两个笑话。都是社交型笑话——安全、无害、大家都能笑的那种。{TA}以前讲的笑话需要想两秒才能懂。”

【裂痕类型二：被忽视】

背景故事：{TA}曾经跟你说过一个{TA}“不知道为什么但就是很准”的预感——关于某件事会发生。后来那件事确实发生了。{TA}提起的时候，你说“巧合吧”。{TA}之后再也没有跟你提过任何“预感”。

S-01特殊旁白：“{TA}看了一眼窗外，说了一句‘可能要下雨’。但天气预报是晴天。{TA}没有解释为什么这么说。{TA}只是在说——然后等你会不会问。”

【裂痕类型三：被表演】

背景故事：{TA}发现自己在你面前变成了一种“角色”——快乐的、轻松的、不需要被照顾的。有一次{TA}真的很难过，但你还是说了“你看起来挺好的”。{TA}从那以后就一直“看起来挺好的”。

S-01特殊旁白：“{TA}今天的妆容比平时精致一些。不是‘变好看了’的那种精致——是‘更完整’了。像是盖上了最后一层。”

游戏启动时执行：
1. 确定场景列表 = [S-01, S-02]

   a. 以 30% 概率决定是否出场
   b. 若出场，在可用空位中随机选择一个位置插入
   c. 从可用空位中移除该位置
5. 添加 S-05/S-06（二选一，50%概率）
6. 添加 S-07 至末尾

function getINFJLine(sceneId, flags) {
  if (sceneId === "S-E1") {
    if (flags["F-02"] === "已触发") {
      return "……我刚才发那段话的时候想撤回。但我想了一下——你说过‘你在我面前不用演’。";
    } else {
      return "……我本来想撤回的。但我还是发了。";
    }
  }
  if (sceneId === "S-E2") {
    if (flags["F-03"] === "已触发") {
      return "……你翻开那本笔记本的时候，我在想你会不会翻到这一页。你翻到了。";
    } else {
      return "……很久以前写的。我当时觉得这是一个我想相信但不太确定会不会发生的事。";
    }
  }
  if (sceneId === "S-E3") {
    if (flags["F-01"] === "已触发" && flags["F-02"] === "已触发") {
      return "……你刚才说‘你在把这件事情讲得很好’——不是‘这件事情本身很好’。你在说我在把它讲好。";
    } else {
      return "……你为什么要说‘你在讲得很好’？那不是‘这件事很好’的意思。";
    }
  }
  return "";
}

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"] && flags["Anchor-06"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
