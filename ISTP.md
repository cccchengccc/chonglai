系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：被入侵】

背景故事：过去半年你频繁在{TA}“做自己的事”的时候打断{TA}——问{TA}在做什么、为什么要做、能不能先陪你。你逐渐让{TA}觉得“独处”是需要被解释的事情。{TA}现在做很多事情都选择在你不在的时候做。

S-01特殊旁白：“{TA}工具箱旁边多了一本新的笔记本——{TA}从来不记笔记。你翻开第一页，写着‘{TA}在家的时候做的事’。{TA}在分类。{TA}在管理你的在场对{TA}活动的影响。”

【裂痕类型二：被情感压迫】

背景故事：你曾经在{TA}专注做事的时候说“你能不能看我一眼”“你能不能回应我一下”。{TA}觉得那种“被要求的情感表达”是一种{TA}无法自然产出的商品。{TA}后来学会了说“嗯”——用最短的声音来满足“回应”的形式要求。

S-01特殊旁白：“{TA}今天修了一把椅子。修完之后没有说‘我修好了’。{TA}把它放回原位，然后看了它一眼。那个动作本身就是一句‘我做了’。只是没有翻译成语言。”

【裂痕类型三：被忽视的付出】

背景故事：{TA}帮你修过很多东西，你大部分时候只说了一句“谢谢”就离开了。{TA}没有抱怨过。但你逐渐发现{TA}不再主动检查哪些东西需要修了——{TA}在等你开口说“这里坏了”。从“主动看到”退到了“被动回应”。

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"] && flags["Anchor-06"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
