四、开局随机背景
系统在游戏开始时随机分配以下三种背景之一，不告知玩家具体类型，仅在 S-01 的旁白中通过特殊细节暗示。

【裂痕类型一：被习惯】

背景故事：过去半年你把{TA}每天的付出当成了“本来就有的”——{TA的}早安、{TA}帮你准备的午餐、{TA}替你记得的事。你没有说“谢谢”的时候越来越多。{TA}发现{TA}在做的“这些事”好像不需要被看见。然后{TA}开始减少——一件一件地减。

S-01特殊旁白：“{TA}今天没有帮你把拖鞋摆正。你的拖鞋是歪的。这个动作以前从来没有发生过。但你可能不会注意到。”

【裂痕类型二：被比较】

背景故事：你曾经无意中说过“某某的伴侣好会照顾人”。{TA}当时笑着点头，但{TA}之后开始更努力地“照顾你”。不是超过别人，是怕你觉得“不够好”。{TA}把自己的标准抬高了。但那个标准现在压着{TA}。

S-01特殊旁白：“{TA}今天准备了比平时更丰富的早餐——多加了一道你随口提过一句的小菜。但你还没有发现那道菜为什么会出现在桌上。因为是{TA}记住的。”

【裂痕类型三：被忽略的存在】

背景故事：你从来没有问过{TA}“你今天需要什么”。{TA}每天在问你需要什么、记住你需要什么、准备你需要什么。但{TA自己}从来没有被问过。{TA}开始觉得“我可能不需要被问”——但这个结论是{TA}替你做出来的。

S-01特殊旁白：“{TA}今天在桌子前面停了一下。{TA}看着你的杯子——{TA}忘了给你倒水。不是故意的。是今天{TA}连问自己都没问。”

function determineEnding(trust, flags, difficulty) {
  if (trust < 30) return "BE-1";
  if (trust >= 30 && trust < 50) return "BE-2";
  if (trust >= 50 && trust < 60) return "BE-3";
  if (trust >= 60 && trust < 70) return "NE-1";
  if (trust >= 70 && trust < 85) return "GE-1";
  if (trust >= 85 && difficulty === "现实线") {
    const count = Object.values(flags).filter(v => v === true).length;
    if (count >= 6 && flags["Anchor-02"] && flags["Anchor-05"] && flags["Anchor-06"]) {
      return "PE-1";
    }
    return "GE-1";
  }
  return "GE-1";
}
